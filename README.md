# Form UI Designs

### Sample designs of Login, Registration, Forgot Password, Change Password and Email Verification Page.

<br/>

## How to Start ?

### Requirements:- 
1> nodejs

<br/>


1. Install all the dependencies
```sh
npm install
```

2. Start the react application
```sh
npm start
```
